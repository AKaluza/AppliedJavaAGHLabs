Collections: Equality contract

Class Person represents selected attributes of real entity

  • Instant created represents a timestamp when instance was created
  • String name represents full name of a person
  • Integer age represents age of a person
  • Address address represent current address where person is living at

Implement equality contract for class Person

  • Two instances of Person class are equal if and only if their fields are equal: name, age, address

Implement equality contract for class Address

  • Two instances of Address class are equal if and only if their fields are equal: street, city.
