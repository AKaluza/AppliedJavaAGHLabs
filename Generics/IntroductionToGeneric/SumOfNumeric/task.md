### Exercise: Sum of Two Numbers (Bounded Type Parameter)

Implement a method sum.

It should:

- Accept two parameters of any numeric type (e.g. Integer, Double, Float),
- Return their sum as a double.

Example:

java
```java
System.out.println(Calculator.add(2, 3));      // 5.0
System.out.println(Calculator.add(1.5, 2.25)); // 3.75

```

Hint: Find superclass for Integer, Double, Float
provide the method doubleValue().
