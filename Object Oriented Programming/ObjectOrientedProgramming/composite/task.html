Object Oriented Programming: Composite

Implement class Employee

  • class must be abstract
  • class must have only two private fields:
    • name of type String
    • salary of type int
  • fields must be set only when new instance is created

Implement class Manager

  • must extend Employee class
  • must implement HavingTeam interface
  • method add should take argument of type Developer
  • should keep all added Developer instances
  • method find should take argument of type String that is a name of developer to find
  • method find should return null if such developer does not exist
  • method find should return reference to Developer instance if such exists

Implement class Developer

  • must extend Employee class
